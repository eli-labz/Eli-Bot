# Eli Bot

Eli Bot is an open-source Windows GUI automation assistant focused on natural-language-driven actions and OS-native interaction.

## Documentation

- Full project documentation: [core/README.md](core/README.md)

## Quick Start

1. Install Python dependencies:

```bash
pip install -r requirements.txt
```

2. Run the launcher:

```bat
assistant.bat
```

## Repository Structure

- `core/` - Main assistant, automation engine, and optional Edge Actions module
- `tests/` - Unit tests
- `requirements.txt` - Python dependencies
